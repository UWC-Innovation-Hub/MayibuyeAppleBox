Project Overview

Interactive Apple Box Archives

Each themed box contains carefully selected artifacts that tell the story of prisoner life on Robben Island. Explore the 3D models with interactive hotspots to discover the historical significance of each item.

About the Project

The UWC Innovation Hub (UIH) has developed four themed interactive Apple Box experiences using the JIGSPACE platform to digitally preserve and present historical artifacts and documents from Robben Island prisoners.

Each themed box features 6-10 carefully selected artifacts with narrative-driven storytelling, historical photographs and printed artwork, and audio narratives providing historical context.

This initiative is part of the broader Unboxing Mayibuye: Access to Digital Heritage Project, aimed at preserving these crucial historical materials for future generations while maintaining their historical significance.
